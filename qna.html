What is DOM and what is the purpose of DOM?

DOM means the Document Object Model.Dom is representation of web structure. It represent the web page as nodes and object. By DOM JS can interact with the web page.

How will you select HTML elements using DOM? Name the DOM methods.

By using document DOM can select html element. There are 5 way to select html document in DOM. They are:-

  1. getElementById()
  2. getElementsByClassName()
  3. getElementsByTagName()
  4. querySelector()
  5. querySelectorAll()

What is event bubble?

Event bubble means event goes up from where a event happened. It always goes up as bubble. Event Bubbling is a concept in the DOM. This is the default behavior of events on elements unless stop the propagation.When an event happens on an element, it first runs the handlers on it, then on its parent, then all the way up on other ancestors.

What is a callback function and why will you use it?

A callback function is a function that is passed as parameter for complete some action. The name of callback function is optional.A JavaScript callback is a function which is to be executed after another function has finished execution.
